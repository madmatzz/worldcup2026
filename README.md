# World Cup 2026 Bracket

A modern, interactive circular bracket for the FIFA World Cup 2026, built with Next.js and React.

## Features

- **Interactive Circular Bracket:** Visualize the knockout stages of the World Cup in a unique circular layout.
- **Modern Tech Stack:** Built using Next.js 15+, React 19, and Tailwind CSS.
- **Vercel Ecosystem:** Integrated with Vercel Analytics and Vercel KV for data storage and performance monitoring.
- **Accessible Components:** Uses Base UI and Radix primitives (via shadcn) for accessible and customizable UI components.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
